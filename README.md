# Test Game

HTML5 → Android APK via Capacitor + GitHub Actions

## 🚀 Быстрый старт
1. Перейдите во вкладку **Actions**
2. Сборка запустится автоматически
3. Скачайте APK из **Artifacts** (~3-5 мин)

## ⚙️ Конфигурация
- **App ID:** `com.testgame.app`
- **Min SDK:** 24 (Android 7.0)
- **Target SDK:** 35 (Android 15)
- **Ориентация:** Альбомная
- **Полный экран:** Да
- **Разрешения:** ACCESS_NETWORK_STATE, INTERNET, CAMERA, RECORD_AUDIO, VIBRATE, ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION, WRITE_EXTERNAL_STORAGE, READ_EXTERNAL_STORAGE, WAKE_LOCK
